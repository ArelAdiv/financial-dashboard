# דשבורד פיננסי אישי 💰

דשבורד פיננסי מקומי עם סוכן AI שמנתח את המצב הפיננסי שלך.

## התקנה והפעלה

### דרישות מקדימות
- Node.js מותקן (הורד מ-https://nodejs.org)

### הפעלה ב-Windows
1. פתח את תיקיית הפרויקט
2. לחץ פעמיים על `start.bat`
3. הדפדפן ייפתח אוטומטית ב-http://localhost:3000

### הפעלה ידנית
```
npm install
node server.js
```
ואז פתח http://localhost:3000 בדפדפן.

## שלבי שימוש

### שלב 1 — אשף הגדרה
בפתיחה הראשונה, המערכת תנחה אותך להזין:
- פרטי משק הבית (בני משפחה, גילאים, הכנסות)
- חשבונות בנק ויתרות
- כרטיסי אשראי
- הלוואות פעילות
- חיסכון, פנסיה, גמל, השקעות

### שלב 2 — טעינת נתונים אמיתיים
לאחר הגדרת הפרופיל, עבור ללשונית "טעינת נתונים":
- ייצא קובץ CSV מהבנק שלך
- גרור אותו לאזור ההעלאה
- תן לו שם מזהה (לדוג': "עו״ש פועלים")
- חזור על הפעולה לכל חשבון/כרטיס

### שלב 3 — הפעלת הסוכן
עבור ללשונית "סוכן AI":
1. הזן את ה-API Key שלך מ-Anthropic (console.anthropic.com)
2. שאל שאלות פיננסיות בעברית חופשית

## ייצוא מהבנקים

| בנק | איפה לייצא |
|-----|------------|
| פועלים | העברות ותנועות ← ייצוא לאקסל |
| לאומי | ניהול חשבון ← תנועות ← ייצוא CSV |
| דיסקונט | חשבון עו"ש ← תנועות ← הורדת נתונים |
| מזרחי טפחות | מידע על החשבון ← תנועות ← ייצוא |
| כאל | כאל דיגיטל ← עסקאות ← ייצוא לאקסל |
| ישראכרט | כרטיסים ← עסקאות ← ייצוא לאקסל |

---

## אבטחה ופרטיות
- כל הנתונים נשמרים **מקומית בלבד** בתיקיית `data/`
- ה-API Key נשמר בדפדפן שלך בלבד (localStorage)
- אין שרת חיצוני, אין שיתוף נתונים
